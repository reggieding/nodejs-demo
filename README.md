## 接口说明
- 运行步骤
  * 1：没有安装nodejs请先安装nodejs。
  * 2：修改请求参数和产品相关密钥设置。
  * 3：命令行运行：node  (具体的js文件名称).js
- 注意事项
  * js文件必须保存为UTF-8编码，否则会报异常。
- 文件说明

```
.
├── textCheckDemo.js 文本在线检测接口演示
├── textCallbakDemo.js　文本离线结果获取接口演示
├── imageCheckDemo.js　图片在线检测接口演示
└── imageCallbackDemo.js 图片离线结果获取接口演示
```
